# AzureVMResumo
Resumo sobre o conteudo disponibilizado pela DIO referente a maquinas virtuais da Azure

Sempre observar o SLA para entender o acordo de nivel de serviços e o tempo de diponibilidade do serviço oferecido.
Na criação das VMs é possivel verificar as zona de disponibilidade.
As constas de armazenamento é possivel verificar estratégias de replicação que podem influenciar na replicação de dados em diferentes lugares ou regiões o que pode afetar/melhorar o SLA. Pode trazer alta disponibilidade. Pode ser que fique mais caro isso. Deve ser analisado se é uma boa solução.
